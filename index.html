📚 Editorial Appointments Map Viewer

📖 How to Use

  1. Select Geocoding Method:
    • ✅ Built-in Database (recommended) - Fast, works offline, geocodes 200+ locations instantly
    • Online Geocoding - Slower (up to 5 minutes), more accurate, may fail due to browser restrictions
  2. Download the Editorial Database CSV file and upload it using the file selector below
  3. Wait for geocoding - Built-in method completes instantly; online method takes several minutes
  4. Filter by Location: Enter a city name in the search box (e.g., "Milan" not "Milan, Italy") and set your desired radius in kilometers
  5. Use Additional Filters: Filter by Journal, Role, Country, or Status to narrow down your results
  6. Find Editors & EAB to Visit! Click on map markers to see full editor details including contact information

💡 Pro Tip: Start with a city name (not country) for more accurate radius searches. Example: Search "Austin" with 100km radius instead of "Texas".

Required columns: Editor, Journal, Role, Country, Affiliation (Email and Website optional)
